🚀 LendSwift — Multi-Step Loan Application Platform
A production-grade, responsive multi-step loan application wizard built with Next.js, React, TypeScript, Tailwind CSS, and Zustand. Designed to simulate a real fintech lending experience — dynamic form flows, real-time validation, and state persistence — end to end.

📖 Overview
LendSwift reimagines the loan application experience for Indian digital lending platforms. Instead of a single overwhelming form, it breaks the journey into an 8-step guided wizard — handling three distinct loan products (Personal, Home, Business) with dynamic field requirements, employment-specific workflows, and a save-and-resume flow so users never lose progress.
